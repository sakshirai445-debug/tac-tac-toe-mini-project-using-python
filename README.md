Description:
------------
This is a Python-based Tic Tac Toe game built using the Tkinter GUI library.
It supports two game modes:
1. Player vs Player
2. Player vs AI (with 3 difficulty levels: Easy, Medium, Hard)

The game includes a modern UI, real-time scoreboard, winner animations, and
a "How to Play" popup for new users.

Features:
---------
- Human vs Human mode
- Human vs AI mode with 3 difficulty levels:
  * Easy   -> AI picks random moves
  * Medium -> AI tries to win or block the opponent
  * Hard   -> AI uses the Minimax algorithm and is unbeatable
- Scoreboard: Tracks wins and draws
- Winning cell animation
- Reset and Main Menu buttons
- Aligned and responsive GUI

Requirements:
-------------
- Python 3.x
- Tkinter (comes with most Python installations)

How to Run:
-----------
1. Ensure Python 3 is installed
2. Navigate to the project folder in a terminal or command prompt
3. Run the script using:
   python main.py

Gameplay:
---------
- The game is played on a 3x3 grid
- Player 1 is 'X' and Player 2 or AI is 'O'
- Each player takes turns placing their symbol
- First to get 3 in a row (horizontal, vertical, or diagonal) wins
- If all cells are filled and no winner, the game ends in a draw
